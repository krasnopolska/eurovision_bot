# 🎶 Eurovision Bot — Інструкція з запуску

## Що потрібно
- Python 3.11+
- Telegram акаунт
- 5 хвилин часу

---

## Крок 1 — Створи бота в Telegram

1. Відкрий **[@BotFather](https://t.me/BotFather)** в Telegram
2. Надішли команду `/newbot`
3. Дай назву боту: `Eurovision 2025`
4. Дай юзернейм: `my_eurovision_bot` (має закінчуватись на `bot`)
5. BotFather надішле тобі **токен** — скопіюй його!

---

## Крок 2 — Налаштуй секрети

Скопіюй шаблон і встав свій токен:

```bash
cp .env.example .env
# відкрий .env і встав BOT_TOKEN
```

Файл `.env` додано в `.gitignore` — він **не** потрапить в git. Не комітити!

---

## Крок 3 — Запусти бота

### Варіант A — Docker (рекомендовано)

Потрібен встановлений [Docker](https://docs.docker.com/get-docker/) + Docker Compose.

```bash
docker compose up -d --build      # запустити у фоні
docker compose logs -f            # подивитись логи
docker compose down               # зупинити
```

База даних SQLite зберігається у Docker volume `eurovision-data` — переживе перебудову контейнера.

### Варіант B — локально без Docker

```bash
pip install -r requirements.txt
export BOT_TOKEN="твій_токен_тут"    # або заповнений .env + `set -a; source .env; set +a`
python bot.py
```

---

## Крок 4 — Стань адміністратором

1. Відкрий бота в Telegram
2. Напиши `/setadmin` — ти станеш адміном
3. Поділись посиланням на бота з друзями!

---

## Крок 5 — Після фіналу Євробачення

Коли стануть відомі результати, введи їх командою:

```
/setresults 1 Швейцарія 2 Франція 3 Ізраїль 4 Швеція 5 Норвегія 6 Австрія 7 Латвія 8 Грузія 9 Вірменія 10 Словенія
```

Бот автоматично підрахує хто найточніше вгадав місця! 🏆

---

## Команди бота

| Команда | Що робить |
|---------|-----------|
| `/start` | Головне меню |
| `/stats` | Загальна статистика |
| `/setadmin` | Стати адміном (один раз) |
| `/setresults 1 Країна 2 Країна ...` | Ввести офіційні результати (тільки адмін) |

---

## Як рахується точність передбачень?

За кожне передбачення ми дивимось, наскільки далеко воно від реального місця:

| Відхилення | Балів |
|-----------|-------|
| 0 (влучно!) | 100 |
| ±1 місце | 80 |
| ±2 місця | 60 |
| ±3 місця | 40 |
| ±4+ місця | 10 або менше |

Фінальна точність = середня кількість балів по всіх передбаченнях.

## Файли проекту

```
eurovision_bot/
├── bot.py             — головний файл бота
├── data.py            — база даних (SQLite)
├── requirements.txt   — залежності

├── .gitignore         — що НЕ комітити (включно з .env та .db)
└── README.md          — ця інструкція

.env                   — твій токен (gitignored, не комітити!)
eurovision.db          — створюється автоматично; в Docker лежить у volume `eurovision-data`
```

## Поширені проблеми

**"ModuleNotFoundError: telegram"**
→ Запусти: `pip install python-telegram-bot==21.5`

**Бот не відповідає**
→ Перевір токен, бот має бути запущений (`python bot.py` в терміналі)

**Помилка прав доступу на /setresults**
→ Спочатку запусти `/setadmin`
